Booklender — Homework 44–47

Используемые технологии
	•	Java
	•	BasicServer
	•	FreeMarker
	•	Библиотека org.json
	•	JSON как файловое хранилище
	•	Cookies (через HTTP headers)
	•	Статические файлы (css, изображения)

⸻

Функциональность

Приложение позволяет:

📚 Работа с книгами
	•	Просматривать список всех книг
	•	Открывать страницу конкретной книги
	•	Просматривать описание и изображение книги
	•	Видеть статус книги
	•	Брать книгу
	•	Возвращать книгу
	•	Ограничение: не более 2 книг одновременно
	•	Нельзя взять уже занятую книгу
	•	Нельзя взять одну и ту же книгу дважды

⸻

👤 Работа с сотрудниками
	•	Просматривать страницу сотрудника
	•	Регистрировать нового сотрудника
	•	Авторизоваться
	•	Выходить из системы

⸻

🔐 Профиль пользователя

После авторизации пользователь может:
	•	Просматривать свой профиль
	•	Видеть информацию о сотруднике (ID и ФИО)
	•	Видеть список текущих книг
	•	Видеть историю всех когда-либо взятых книг

Профиль недоступен без авторизации.

⸻

📂 Работа с JSON
	•	Данные книг загружаются из books.json
	•	Данные сотрудников загружаются из employees.json
	•	Данные читаются при каждом запросе
	•	Добавление новых записей в JSON не требует перезапуска сервера

⸻

Структура данных

Book
	•	id
	•	title
	•	author
	•	image
	•	description
	•	status

⸻

Employee
	•	id
	•	fullName

⸻

Хранение данных во время работы сервера

Используются структуры:
	•	Map<String, Employee> — зарегистрированные пользователи
	•	Map<String, String> — пароли
	•	Map<String, String> — активные сессии
	•	Map<String, List> — текущие книги пользователя
	•	Map<String, List> — история книг
	•	Map<Integer, String> — занятые книги

⸻

Ограничения системы
	•	Максимум 2 книги одновременно
	•	Нельзя взять книгу, если она уже занята
	•	Нельзя вернуть книгу, если пользователь её не брал
	•	Профиль доступен только после авторизации
	•	Данные не сохраняются обратно в JSON (изменения хранятся в памяти сервера)

⸻

Как запустить проект
	1.	Убедиться, что подключены библиотеки:
	•	FreeMarker
	•	org.json
	2.	Запустить класс Main.
	3.	Открыть браузер.
	4.	Перейти по адресу:
http://localhost:9889/books

Если сервер запускается корректно, в консоли появится сообщение о старте сервера.

⸻

Если возникают ошибки

Ошибка FreeMarker (переменная null)

Необходимо проверить:
	•	что в handler передаётся модель через Map
	•	что используется правильный ключ (books, book, employee, currentBooks, pastBooks)

⸻

Браузер пишет, что localhost ничего не отправил

Проверить:
	•	что сервер запущен
	•	что порт указан правильно
	•	что id существует в JSON
	•	что шаблон лежит в папке data

⸻

JSON не читается

Проверить:
	•	путь data/books.json и data/employees.json
	•	корректность формата JSON
	•	подключена ли библиотека org.json

⸻

JSONArray / JSONObject подсвечиваются красным

Значит библиотека org.json не подключена к проекту.
