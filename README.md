Booklender — Homework 44

Используемые технологии
	•	Java
	•	BasicServer
	•	FreeMarker
	•	Библиотека org.json
	•	JSON как файловое хранилище
	•	Статические файлы (css, изображения)

Функциональность

Приложение позволяет:
	•	Просматривать список всех книг
	•	Открывать страницу конкретной книги
	•	Просматривать страницу сотрудника
	•	Загружать данные из JSON без перезапуска сервера при добавлении новых записей

Данные не изменяются. Реализовано только чтение.


Структура данных

Book
	•	id
	•	title
	•	author
	•	image
	•	description
	•	status

Employee
	•	id
	•	fullName

Как запустить проект
	1.	Убедиться, что подключены библиотеки FreeMarker и org.json.
	2.	Запустить класс Main.
	3.	Открыть браузер.
	4.	Перейти по адресу:
http://localhost:9889/books

Если сервер запускается корректно, в консоли появится сообщение о старте сервера.

⸻

Если возникают ошибки

Если возникает ошибка FreeMarker (переменная null), необходимо проверить, что в handler передаётся модель данных через Map и используется правильный ключ (books, book, employee).

Если сервер не отвечает или браузер пишет, что localhost ничего не отправил, необходимо проверить:
	•	что сервер запущен
	•	что порт указан правильно
	•	что id существует в JSON
	•	что шаблон лежит в папке data

Если JSON не читается, необходимо проверить:
	•	путь data/books.json и data/employees.json
	•	корректность формата JSON
	•	подключена ли библиотека org.json

Если классы JSONArray или JSONObject подсвечиваются красным, значит библиотека json не подключена к проекту.
